Troubleshooting
===============

Shapely error
-------------

If you recieve an error like this:

.. code::

    P3917: ERROR:shapely.geos: Geometry must be a Point or LineString (2021-03-16 09:48:37; geos.py:252)
    ...more stuff...
    Aborted (core dumped)
    
You need to reinstall shapely using the ``--no-binary`` option. 
With your Detcap virutal environment active, run:

.. code::

    pip uninstall shapely
    pip install shapely --no-binary shapely
