Detcap Documentation
====================

Detcap is a tool for monitoring the detection capability of seismic networks.
It achieves this by performing calculations on amplitude and data latency
read from seismic stations and puts the resulting values in a geospatial
context. 

**If you're here looking for documentation on the website UI, head to**
:ref:`user-interface`.
